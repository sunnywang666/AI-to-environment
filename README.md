# 云的身体 · The Body of the Cloud

一个关于 **AI 对环境与能源消耗** 的数据新闻可视化长卷（scrollytelling）。

> 你问了 AI 一个问题，有一件看不见的事正在发生。
> 把它乘以 25 亿次，就是一个国家级的胃口。

## 项目简介

数据新闻课程作业。通过 6 段因果链叙事，把 AI「干净、无形」的使用体验，
还原成它在现实世界里真实的电、水、碳与土地代价，并对照中美两国的情况。

- 艺术方向：**热成像 · 墨黑沉浸**（Making the Invisible Physical）
- 形式：纯静态网站，scroll-driven 沉浸式叙事
- 技术：HTML/CSS + Scrollama.js + D3.js（无构建工具，CDN 引入）

## 叙事结构

1. 一次点击 — 打破 AI 的"干净感"
2. 一个国家的胃口 — 量级反转
3. 云有一个邮编 — 把弥散的云钉到具体经纬度（中美双线）
4. 账单 — 谁在为此买单（情绪重心）
5. 蒙眼布 — 同一件事的数字相差千倍（问责脊梁）
6. 这笔交易公平吗 — 三个杠杆与个人能动性
