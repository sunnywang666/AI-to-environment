> 配套叙事方案：`AI能源代价_重构叙事方案_v2(1).md`
> 艺术方向：**热成像 · 墨黑沉浸**（Making the Invisible Physical）
> 气质：纪录片 / 沉浸 / 克制诗意 —— 不末日、不洗白、不 PPT

## 0. 核心概念

整个网站是一台「显形机器」。AI 的"干净感"是设计出来的——没有烟囱、没有排气管，只有一个旋转小圆点。
视觉从**冷、空、数字化**（模仿 AI 干净 UI）开始，随着代价揭开，逐步变**物理、有质感、有温度**：卫星颗粒 → 等高线 → 热力渐变 → 真实变电站照片。
**向下滚动 = 幻觉被一层层剥开。**

**贯穿动机（through-line glyph）：那个加载小圆点会变形**
| 段落 | 形态 |
|---|---|
| 1 一次点击 | 无辜的旋转 spinner `·` |
| 2 国家的胃口 | 电表指针 |
| 3 云有邮编 | 地图定位点 |
| 4 账单 | 水滴 |
| 5 蒙眼布 | 带误差棒的模糊光斑 |
| 6 公平吗 | 回到 spinner，但慢下来 |

## 1. 色彩 Tokens

```css
:root {
  /* 底色 */
  --ink:        #0C0E11;  /* 墨黑，沉浸段主背景 */
  --ink-2:      #14171C;  /* 卡片/分层 */
  --paper:      #F4F1EA;  /* 暖纸白，编辑文字段（与墨黑交替）*/

  /* 文字 */
  --text:       #ECEAE3;  /* 暗底正文 */
  --text-dim:   #8A8E96;  /* 注释/来源 */
  --text-ink:   #1A1C20;  /* 纸白段正文 */

  /* 三维度编码（全片复用，图例固定）*/
  --energy:     #F5B731;  /* 电 · 电流琥珀 */
  --water:      #3FB6C8;  /* 水 · 冷青 */
  --carbon:     #2A2D34;  /* 碳 · 石墨（暗底上用浅描边 #4A4E57）*/

  /* 警示——极度克制，只在第4段「人的代价」出现 */
  --ember:      #E5533C;  /* 余烬红：断电、欠费、人物 */

  /* 数据状态区分 */
  --measured:   solid;    /* 实测 = 实线/实心 */
  --projected:  rgba(245,183,49,.45); /* 预测 = 半透明/虚线 */
}
```

**用色纪律**
- 余烬红 `--ember` 全片出现不超过 3 处，全部在人物/账单语境。滥用即失效。
- 三维度色在任何图表、图例、行内强调中**含义恒定**：黄=电、青=水、石墨=碳。
- 预测值永远比实测值"虚"：虚线、半透明、或加 `~` 前缀。

## 2. 字体

| 用途 | 字体 | 字重 |
|---|---|---|
| 中文标题 | 思源黑体 Source Han Sans | Heavy / 900 |
| 正文（人物故事、叙事） | 思源宋体 Source Han Serif | Regular |
| 大数字 / 数据标签 | Roboto Mono 或 Söhne Mono（等宽/紧缩）| Medium |

```css
--font-display: "Source Han Sans SC", "思源黑体", sans-serif;
--font-serif:   "Source Han Serif SC", "思源宋体", serif;
--font-mono:    "Roboto Mono", ui-monospace, monospace;
```

**纪念碑大数字**：`448 TWh`、`$940.08`、`519ml` 这类关键数字做成超大字（桌面 ≥ 96px），滚动进入时 **count-up 动画**累加。数字用 mono，单位用小一号。

## 3. 运动语言

- **慢、克制、scroll-driven。** 缓动用 ease-out，禁止弹跳/回弹/花哨。
- 数字滚动累加；地图分层淡入；图层透明度过渡。
- 气质参照 Pudding《Satellites》：图 sticky 不动，文字滚动，图随滚动进度变化。
- 进出场用透明度 + 轻微位移（≤24px），不要大幅平移。

```css
--ease: cubic-bezier(0.22, 1, 0.36, 1);
--dur:  600ms;
```

## 4. 布局系统

**两种段交替，张弛有度：**
1. **沉浸段**（墨黑全幅）：`position: sticky` 图形 + 滚动文字。地图、滑块、卫星 before/after 用这种。
2. **编辑段**（暖纸白窄栏）：阅读宽度 `max-width: 680px`，宋体正文，留白多。人物故事、因果拆解用这种。

```css
--reading-width: 680px;
--gutter: clamp(20px, 5vw, 80px);
```

**移动端**：双联地图 → 上下堆叠；横向滑块 → 改为点击切换步骤。或明确标注"建议桌面浏览"。

## 5. 落地优先级（防范围爆炸）

6 段全程交互做不完会全盘垮。**集中资源做 2 个明星交互，其余降级为滚动触发静态动画。**

| 优先级 | 模块 | 形态 |
|---|---|---|
| ★ 明星 | 第1段 开场估算器 | **查表估算**（预设查询类型×系数，标注"估算值"），不接真模型 |
| ★ 明星 | 第5段 口径滑块 | 冷却→+发电→+芯片全生命周期，数字 0.3ml→519ml 跳变 |
| ◆ 重点 | 第2段 虚拟国家用电排行 Hero 图 | D3 条形，count-up |
| ◆ 重点 | 第3段 中美双联地图 + 水压力图层 | MapLibre / SVG |
| ○ 降级 | 第3段 卫星 before/after 滑块 | 静态对比图 + 拖动条 |
| ○ 降级 | 第4段 多因素拆解、双人物卡 | 滚动触发静态信息图 |
| ○ 降级 | 第6段 杰文斯悖论双轴折线 | D3 静态 + 滚动高亮 |

## 6. 技术栈

纯静态、无后端、可直接部署（GitHub Pages / Vercel）：
- `Scrollama.js` —— 滚动触发
- `D3.js` —— 图表 / 数据地图
- `MapLibre GL` 或纯 SVG —— 地理底图
- `GSAP`（可选）—— 复杂时间线动画

```

├─ css/  tokens.css  layout.css
├─ js/   scroll.js  charts.js  estimator.js  slider.js

```

## 7. 严谨性红线（评审会盯的点）

1. 实测 vs 预测 **视觉必须可区分**（实线/虚线）。
2. 第4段电费暴涨做**诚实多因素拆解**——数据中心是"增长最快的新增需求"，不是唯一原因。
3. 第5段 1000 倍差异定性为**系统边界不同**，不是"谁撒谎"。
4. 每个数字旁可点开来源。交稿前逐条复核链接有效。
